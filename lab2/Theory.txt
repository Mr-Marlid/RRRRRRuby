1) Объекты это экземпляры класса (class), для того чтобы манипулировать объектом - сперва нужно создать класс этого объекта.
class Human #создаем класс, присваиваем ему имя Human
    attr_accessor :name, :age # аксессоры

    def initialize (name, age) # конструктор класса.
        @name = name # переменная объекта
        @age = age
    end
end

qpi = Human.new('Qpi', 32) # создаем объект qpi
puts "Меня зовут #{qpi.name} и я родился #{qpi.age} года назад "

2)Инкапсуляция - это принцип сокрытия информации, путем управления областью видимости атрибутов и методов класса.


def name # геттер
    @name
end
def name=(name)  # сеттер
    @name=name
end

а если приспичит, создать в классе метод, который будет это делать.

Аксессоры бывают 3х типов, аксессор только на чтение - attr_reader, 
аксессор только на изменение/запись attr_writer 
и аксессор на чтение и изменение attr_accessor.
Объявим аксессор на чтение и изменение атрибутов, перечисленных через запятую.
attr_accessor :name, :age # аксессоры
